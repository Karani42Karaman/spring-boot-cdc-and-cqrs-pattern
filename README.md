## Prerequisites

* Java 11
* Maven 3.3+
* Docker 19.03+
* Docker Compose 1.25+

## Installation

```sh
sudo chmod +x build.sh
```

```sh
./build.sh
```


```sh
docker-compose up -d 
```

```sh
cd debezium-connector
```

```sh
sudo chmod +x connector.sh
```

```sh
./connector.sh
```
